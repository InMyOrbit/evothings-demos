Vanilla SMS App

The purpose of this app is to show the functionality of the Cordova SMS Plugin.

Enter a mobile number and a message, thereafter press the Send SMS button. On iOS the Messages app will be opened, and on Android the message will be sent instantly.

To modify this example, follow these simple steps:

  1. Press the "Code" button in the Evothings Workbench window to see the source file of this example app.
  2. Open the file "index.html" in a text editor.
  3. Make some changes and save the file.
  4. Voilà! If everything works, the app automatically reloads!
